# Открой Красноярск
Веб-сайт туристического информационного центра Красноярска, созданный для удобного знакомства с природными объектами города и планирования прогулок.

## О проекте
Сайт помогает пользователям выбрать место для отдыха, узнать информацию о туристических объектах и посмотреть особенности каждого маршрута.
Проект разработан в рамках производственной практики.

## Возможности
- Главная страница с презентацией объектов
- Интерактивная карта с достопримечательностями
- Карточки туристических объектов
- Страницы отдельных объектов:
  - Серебряный лог
  - Николаевская сопка
  - Остров Молокова
- Адаптивная верстка под разные устройства
- Анимации и интерактивные элементы

## Используемые технологии
- HTML5
- CSS3
- JavaScript
- Figma (разработка дизайна)
- Git / GitHub
